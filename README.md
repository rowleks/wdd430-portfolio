# wdd430-portfolio

Personal portfolio built with Next.js and Tailwind CSS.

**Live:** [wdd430-portfolio-rho.vercel.app](https://wdd430-portfolio-rho.vercel.app/)

### Installation

```bash
git clone https://github.com/rowleks/wdd430-portfolio.git
cd wdd430-portfolio
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see it locally.
